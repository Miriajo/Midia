# Midia - Educational project

This project is for educational porpouse only.

## Before to execute

You need to insert your Google API Key into the Info.plist file with name "apiKey" to be able to work with Books section.

## Characteristics

There are 4 sections in the tab bar:

1. Recents - results for Books or Movies base on the code.
2. Search - finder
3. Favorites - Where favorites are storaged
4. Bookmarks - Here the user can switch between Books or Movies.
